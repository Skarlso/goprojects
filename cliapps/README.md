Test Flags
==========

Usage:
```go
go run go_cli.go --help
```

To provide a flag:
```go
go run go_cli.go --name=YourName
```

Also accepts defaults if no name is provided:
```go
go run go_cli.go
```

Will out put:
```bash
Your provided name is: greg
```
